# EmbersUELPeripheryFixx

This is a mod for Minecraft 1.12.2 designed to resolve a crash issue between Thaumic Periphery and Embers UEL on server versions.

## Features
- Fixes compatibility issues between Thaumic Periphery and Embers UEL.
- Ensures server stability when both mods are installed.

## Installation
1. Download the latest release of the mod from the `build/libs` directory.
2. Place the `peripheryfix-1.0.0.jar` file into the `mods` folder of your Minecraft server.
3. Start the server to apply the fix.

## Compatibility
- Minecraft Version: 1.12.2
- Mods: Thaumic Periphery, Embers UEL
